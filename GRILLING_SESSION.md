# Sessão Grill-Me — Plano do Projeto

## Visão Geral
Site-portfólio interativo para prospecção de clientes, demonstrando o funcionamento de landing pages e a importância de cada seção.

## Stack
- Next.js + React + TypeScript
- Deploy: Local apenas
- Prazo: 15 dias

## Estrutura do Site
- **Header fixo:** Logo | Abas (3 modelos) | Botão "🎓 Modo Explicação" liga/desliga
- **Abas horizontais no topo:** alternam entre os 3 modelos na mesma página
- **Tour guiado na primeira visita:** overlay escuro + tooltips flutuantes + botão "Próximo"
- **Tooltips:** explicam o propósito de cada seção e de cada elemento dentro dela
- **Responsividade:** total (mobile, tablet, desktop)

## Os 3 Modelos

| Modelo | Estilo Visual |
|---|---|
| Médicos/Dentistas | Clássico/Sóbrio |
| Advogados | Clean/Minimalista |
| Corretores de Imóveis | Moderno/Arrojado |

## Seções de Cada Modelo
1. **Hero** — título + subtítulo + CTA
2. **Métricas animadas** — números crescentes do 0 até o máximo (ex: +100 imóveis vendidos, 100% satisfação)
3. **Sobre Nós**
4. **Serviços / Especialidades**
5. **Depoimentos / Prova social**
6. **Footer** com contato

## Comportamento
- **Abas horizontais no topo:** alternam entre os 3 modelos na mesma página
- **Modo Explicação (liga/desliga):** botão no header
- **Tour guiado na primeira visita:** overlay escuro + tooltips flutuantes + botão "Próximo"
- **Tooltips:** explicam o propósito de cada seção e de cada elemento
- **Métricas animadas:** números crescentes do 0 até o valor máximo
- **Responsividade:** total (mobile, tablet, desktop)
- **Dados:** fictícios
- **Deploy:** local
